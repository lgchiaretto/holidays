Holidays API

Sistema completo de gestão de feriados nacionais, estaduais e municipais do Brasil com API RESTful e autenticação JWT.

Login Usuário Login Admin Documentação API

Feriados Brasileiros

Cadastro completo de feriados nacionais, estaduais, municipais e opcionais com suporte ao formato brasileiro DD/MM/AAAA.

Autenticação JWT

Sistema seguro de autenticação com tokens JWT, controle de roles (admin/usuário) e proteção de endpoints.

API RESTful

API completa com endpoints para CRUD de feriados, gestão de usuários e documentação Swagger interativa.

SQLite Database

Armazenamento leve e eficiente com SQLite, ideal para containers e deployments em Kubernetes/OpenShift.

Container Ready

Imagem Docker otimizada com Red Hat UBI, pronta para deploy em OpenShift, Kubernetes ou Podman.

PatternFly UI

Interface moderna baseada no design system PatternFly da Red Hat, com experiência consistente e profissional.
